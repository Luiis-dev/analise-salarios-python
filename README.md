# 📊 Análise de Salários com Python

> Projeto de **análise de dados** explorando salários na área de tecnologia, utilizando Python e Jupyter Notebook.

---

## 📈 O que foi analisado

- 💰 Distribuição de salários por área/cargo
- 📍 Variação salarial por região
- 🔍 Tendências e insights do mercado de tech
- 📊 Visualizações gráficas dos dados

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| Python 3 | Linguagem principal |
| Jupyter Notebook | Ambiente de análise |
| Pandas | Manipulação de dados |
| Matplotlib / Seaborn | Visualização de dados |

---

## 🚀 Como rodar

```bash
git clone https://github.com/Luiis-dev/ANALISE-SAL-RIOS-PYTHON.git
cd ANALISE-SAL-RIOS-PYTHON

# Instale as dependências
pip install pandas matplotlib seaborn jupyter

# Inicie o Jupyter
jupyter notebook
```
